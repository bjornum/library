Width, Height, Padding, Borders, Margin and how to adjust after adding these

Width and height

width is how long it is, and height is how tall the element are
it need a value afterward which describe the amount it will affect the element.
one example are width: 100px;

Padding

Padding will increase the "air" inside of the element itself. making it expand
easily spotted if having an box
by doing an padding: 10px; you will see that it will be 10 px on every side of the border, making it expand

border

Border will surround the element, creating a "wall" around it
also modified by border: 10px; creating how big it is

margin

Margin is the air outside the element and defines distance to other elements
one example is margin-bottom: 10px; as this will push the element under 10px away

Adjusting to fit the element to the picture

By this i mean: if originally having an width:200px; and height:80px;
then by adding additional modifiers as padding, border or margins will change the size and position
So if wanting to get it back to how it were, WITH the additional modifiers.
you will have to substract the added modifiers from the value you want to adjust.
wherever it is the width, height or both

Text-align

Text-align defines the horizontal positioning of a text, within an element
it got 4 elements to it, and will stay in its position as you zoom in or out
{text-align: center;} will make the text go to the center
{text-align: left;} or {text-align: right;} will do as it say
{text-align: justify;} im not sure about yet
